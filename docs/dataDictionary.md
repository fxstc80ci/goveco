# Data Dictionary

## Common fields

| Field | Description |
|---|---|
| entity_id | Stable unique entity identifier. |
| parent_id | Immediate parent entity identifier. |
| ultimate_parent_id | Highest known controlling parent. |
| jurisdiction | Country, province, territory, municipality, or Indigenous jurisdiction. |
| fy | Fiscal year. |
| source_id | Stable source identifier from the source registry. |
| confidence | Confidence rating for the row. |
| extracted_at | Date/time the value was extracted. |

## Core measures

| Field | Description |
|---|---|
| gross_entity_spend | Total expense reported by the entity. |
| direct_taxpayer_cost | Direct tax or debt-supported cost. |
| citizen_funded_revenue | Revenue collected from users, ratepayers, consumers, businesses, or regulated parties. |
| net_public_burden | Estimated consolidated burden after classification and offsets. |
| public_balance_sheet_exposure | Balance-sheet or contingent public exposure. |
| net_unrecovered | Unrecovered public capital after repayments, remittances, impairments, or write-offs. |
